# **Windows系列镜像收费策略调整**

为进一步优化成本管控，规范Windows镜像激活与计费流程，平台计划对快杰-o2系列云主机、GPU-高性价比显卡7机型的Windows镜像许可证费用进行调整。

具体调整内容如下：

- **影响机型：**快杰o2系列机型 及 GPU-高性价比显卡7机型

- **生效范围与时间：**部分可用区  26年4月3日起

- **变更范围：**

  - o2系列快杰云主机、GPU-高性价比显卡7机型，在部分地域下使用启用平台许可证的Windows镜像时，将新增许可证费用。
 
  - 启用平台许可证的windows镜像：
    
    - **基础镜像**中提供的全部windows镜像。
      
    - 由使用基础镜像的主机制作的**自制镜像**。
      
    - 导入的自定义镜像且**选择使用平台许可证**。（导入选择自带许可证的自定义Windows镜像创建主机后将不会自动激活，**不激活机器可能出现定期黑屏或重启的情况**，请注意！）
   
  - **存量影响：**对于符合上述影响范围的存量实例，仅在执行涉及计费的操作（包括但不限于重装系统、实例改配、订单续费）时，才会新增该Windows镜像平台许可证费用；未执行上述计费操作的存量实例，仍按原有规则正常使用，不新增任何相关计费。

  - **计费变更：**
    
    - 镜像许可证费用单位以CPU核数为准。
      
    - 具体收费标准以控制台实际显示为准，许可证计费时长与对应实例规格的计费时长保持一致，重装、改配时对应费用会有增减。
      
    - 镜像费用将与主机费用同步收取，生成一条订单信息。
   


# **云主机产品机型上新 - 快杰O型系列拆分上新**

本次变更为正常产品迭代，变更涉及以下调整：

- 机型：快杰O机型

- 可用区：全地域

- **变更内容：**

  - 快杰 O 系列**部分可用区已上线新一代机型**，服务器搭载了Intel 至强第五代处理器（**EmeraldRapids**），提供更强大的计算能力和更快的内存速度。

  - 快杰 O 系列机型将拆分规格族售卖，规格族与 CPU 平台对应关系如下： 

    | 规格族  | CPU厂商            | CPU平台                                                      | 与原机型对比       |
    | ------- | ------------------ | ------------------------------------------------------------ | ------------------ |
    | o1i     | Intel (x86_64)     | Intel CascadeLake: 2.5GHz *\|* 3.9GHz<br />Intel CascadelakeR: 3.0GHz *\|* 4.0GHz<br />Intel IceLake: 2.6GHz *\|* 3.4GHz |                    |
    | o1a     | AMD (x86_64)       | AMD EPYC2: 2.9GHz *\|* 3.4GHz                                |                    |
    | o1r     | Ampere（AArch64）  | Ampere Altra: 3.0GHz                                         |                    |
    | **o2i** | **Intel (x86_64)** | **Intel EmeraldRapids: 2.7GHz *\|* 4.0GHz**                  | **更新一代处理器** |

    详细信息请参考[文档](/uhost/introduction/uhost/type_new)。

    相关API变更请参考[文档](/api/uhost-api/index)。

  - 特性变更：

    - 不同规格族支持网络增强2.0特性的镜像不完全相同。在**o1系列支持**网络增强2.0特性，但在**o2系列中不再支持**的基础镜像如下：
      - Ubuntu：
        - 18.04 64位（高内核、快杰）
        - 16.04 64位（高内核、快杰）
        - 14.04 64位（高内核、快杰）
      - CentOS：
        - 6.x 64位（高内核、快杰）
      - Debian：
        - 10.0 64位
    - 特性相关信息请参考[文档](/uhost/introduction/feature/feature)。

  - 同可用区下，不同规格族的实例价格可能存在不同，具体价格请参考控制台。

> 限制生效时间和范围：2026年3月13日起  全地域可用区
