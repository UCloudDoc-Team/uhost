# 优化DNS配置方法

## Step 1  配置冗余DNS Server地址

可防止DNS Server单点故障后，域名无法解析的情况。

系统中选择control panel(控制面板) -\> Network and Sharing Center (网络与共享中心) -\>
选择当前网络连接 （如Ethernet 2）-\> Properies(属性)-\>双击Internet Protocol
Version 4 (TCP/IPv4)

Server（备用DNS服务器）中，按机房，配置下面表格中的两个IP。
